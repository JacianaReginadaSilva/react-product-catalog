# 📦 Catálogo de Produtos - Unidade 2

Projeto desenvolvido como parte da Atividade Prática da Unidade 2 do curso de **Análise e Desenvolvimento de Sistemas**.

## 🚀 Funcionalidades
- **Listagem Dinâmica:** Exibição de produtos a partir de um array de dados.
- **Componentização:** Uso de componentes reutilizáveis para os cards de produtos.
- **Destaque de Promoções:** Lógica condicional (operador ternário) para destacar ofertas.
- **Cálculo de Total:** Uso do método `reduce` para somar o valor total do catálogo.
- **Desafio Extra:** Formulário funcional com `useState` para cadastrar novos produtos em tempo real.

## 🛠️ Tecnologias
- React
- Vite
- JavaScript (ES6+)
- CSS Inline

## 🏁 Como rodar o projeto
1. Clone o repositório
2. Instale as dependências: `npm install`
3. Rode o servidor: `npm run dev`
